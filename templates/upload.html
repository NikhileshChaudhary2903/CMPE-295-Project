How it works

The Process

Wallet Creation

Creation of wallets for transacting files.

File Partition and Consensus

Client gives the files which is divided into chunks and P2P network decides which node would be selected as the provider node for storing the chunks using a consensus algorithm.

Final Step

File chunks are successfully stored on the provider nodes along with metadata which would help us to retrace the file back using the wallet and the file name.
